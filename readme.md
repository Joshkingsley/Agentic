# 🌍 Arbitrage Intelligence Engine (AIE)

> **An autonomous, deterministic market-intelligence platform designed to exploit cross-border and regional spot-price discrepancies by treating physical commodities as tradable arbitrage assets.**

---

## 🚀 Overview

Traditional dropshipping and global sourcing rely on guesswork and fragmented supply chains. The **Arbitrage Intelligence Engine (AIE)** replaces intuition with pure mathematics. Operating through a bidirectional framework, AIE serves two primary market segments:
1. **Micro-Importers:** Optimizing imports from the United States to Kenya by scanning liquidation web data for high-margin retail spreads.
2. **Enterprise Cooperatives:** Empowering local agricultural and retail cooperatives to route physical assets optimally (e.g., Kirinyaga to Nairobi) and export to premium international storefronts.

By leveraging an autonomous multi-agent pipeline, AIE harvests real-time pricing data, strips out calculation variance using hardcoded logistical math, and aggregates small-scale capital to bypass institutional Minimum Order Quantities (MOQs).

---

## ✨ Core Features

* 📊 **Live Trade Signals (High-Fidelity Terminal):** Real-time data feeds powered by scraping agents that monitor global liquidation indexes versus local Kenyan market prices, automatically calculating guaranteed margin splits.
* 💰 **Capital Pooling Protocol:** A decentralized entry system that aggregates financial resources from smaller startups to collectively clear MOQs, unlocking wholesale pricing tiers that were previously inaccessible.
* 🗺️ **Spatial Logistics Router:** A predictive routing tool that analyzes real-time local logistics, fuel variance, and supply chain bottlenecks to calculate the exact tipping points for volume scaling.
* 🎙️ **Audio Intelligence Briefs:** High-fidelity, synthesized audio trade briefings capturing optimal trade signals, designed for executive-level, on-the-move intelligence.

## 🛠️ Technical Architecture

AIE is built for extreme speed, low-latency AI processing, and zero-variance mathematical accuracy.

### Frontend
* **Framework:** React (Bootstrapped with Vite for instant HMR)
* **Styling:** Tailwind CSS for a premium, cinematic UI/UX
* **Deployment:** Render

### Backend & AI Processing
* **Core Engine:** Django / Python (Deterministic math and API routing)
* **Data Harvesting:** Bright Data Web Unlocker & Scraping Browser 
* **LLM Orchestration (via Groq):** * `openai/gpt-oss-120b` for heavy architectural logic
  * `llama-3.3-70b-versatile` for lightning-fast schema extraction and multi-agent coordination
* **Voice Synthesis:** ElevenLabs API

---

## 🧮 The Arbitrage Equation

At the core of AIE is a strict, hardcoded profit calculation ensuring absolute precision:

`Net Margin = Local Retail Value - [USD Spot Price + Freight Weight Cost + Tariff Constants]`

*Unlike standard LLM calculations which suffer from variance, AIE passes cleanly parsed data into this deterministic Django backend equation, guaranteeing institutional-grade accuracy.*
